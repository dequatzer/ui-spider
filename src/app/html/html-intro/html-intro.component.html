Welcome to HTML tags

HTML :

HTML stands for Hyper Text Markup Language. It is used to create web pages. It describes the structure of the web page. A web page generally consists of different parts like header, body, footer etc. It is used to define these parts of the web pages. This is done using the HTML elements.

HTML elements :

HTML elements are the building blocks of HTML pages. There are elements to define the header, title, body, footer etc.


Tags :

The elements in HTML are represented with the help of tags. There is a tag to define each and every part of a webpage.


Syntax :

<tagName> …some content here... </tagName>

Below is the simplest HTML page that can be created

<!DOCTYPE html> <html> <head> <title>A HTML Page</title> </head> <body> My first HTML page. </body> </html>

HTML pages are saved with .html extension and are executed by browsers. There are lots of browsers like IE (Microsoft Internet Explorer), Mozilla Firefox, Google Chrome, Safari etc. When an html page is executed by the browser, the content within the tags are rendered and displayed in the browser. Tag themselves are not displayed. Learning HTML is very easy as it does not require any special software, a simple Notepad and a browser will do the job. Try saving the above code in a note pad, save it as “mypage.html” and open it with a browser. You just created an HTML page.


Attributes:

A tag can have one or more attributes. Attributes are basically name-value pairs, separated by = and they are written within the start tag of an element after the element’s name. <tagName attrName=’value’> …some content …</tagName> These attributes are like properties that a tag can have. Pick a tag and you can check the attributes list it contains. <div id=”content”>….some text here ….</div> Here “id” is an attribute, which has a value equals to “content”. As you progress you will understand what the purpose of these attributes is and why they are important.
